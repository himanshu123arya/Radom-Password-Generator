# Random Password Generator

A modern, responsive web app to generate secure random passwords with customizable options.

## Features
- Choose password length (6-32)
- Toggle lowercase, uppercase, numbers, symbols
- Copy password to clipboard
- Password strength indicator (weak, medium, strong)
- Clean, professional UI with icons

## File Structure
```
password-generator/
├── index.html        # Main HTML file
├── style.css         # Styles for modern responsive UI
├── script.js         # Password generation logic
└── README.md         # Project instructions
```

## How to Use
1. Open `index.html` in your browser.
2. Adjust password length and character options.
3. Click **Generate Password**.
4. Copy the password using the copy button.
5. Check the strength indicator for password quality.

## Customization
- Change colors in `style.css` for your own theme.
- Adjust password rules in `script.js` as needed.

## Explanation
- **index.html**: Structure and UI elements, links CSS/JS, uses Font Awesome for icons.
- **style.css**: Modern card layout, gradients, spacing, responsive design, color-coded strength indicator.
- **script.js**: Handles password generation, option toggles, clipboard copy, and strength calculation.

## Step-by-Step Instructions
1. Download or clone the project folder.
2. Open `index.html` in any modern browser.
3. Use the sliders and checkboxes to set your password preferences.
4. Click **Generate Password** to create a new password.
5. Click the copy icon to copy the password to your clipboard.
6. View the password strength indicator for feedback.

Enjoy secure, customizable passwords with a beautiful UI!
